Вводим команду для перехода к следующему уровню

>>su level02
>>f2av5il02puano7naaf6adaaf (токен полученный на предыдущем уровне)

Переходим по пути 

>>cd ~
>>ls 

Копируем найденный файл по ssh из виртуальной машины, для этого пишем команду из терминала основной сессии

>>scp -P 4242 level02@192.168.0.11:~/level02.pcap .
>>f2av5il02puano7naaf6adaaf

Скачиваем программу для открытия файлов pcap Wireshark, открываем файл через нее Follow->TCP Stream
Находим пароль ft_wandr...NDRel.L0L 
Т.к. буквы повторяются после точек делаем вывод, что точка указывает на замену символа

Вводим пароль

>>su flag02
>>ft_waNDReL0L

Запускаем команду 

>>getflag
